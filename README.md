# exercicio_3dados
Considerando o lançamento aleatório de três dados representados pelo código a seguir:

import random

d1 = random.randint(1, 6)
d2 = random.randint(1, 6)
d3 = random.randint(1, 6)

print(f'd1: {d1}')
print(f'd2: {d2}')
print(f'd3: {d3}')
Mostre ao usuário qual deles gerou o maior valor.
